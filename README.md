# AlanCartridge
Axiometa x Anthropic 'Jul 2026 Hackathon Project

```
INFINITE CARTRIDGE forge
  python main.py --theme "crabs in space"        generate one game and flash it
  python main.py --loop                          forge next on every game-over
  python main.py --pregen 5                      pre-generate 5 games
  python main.py --flash library/%game_name%     flash a game
  python main.py --list                          cartridge catalogue
```
